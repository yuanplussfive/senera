# ToolSearchTool

## 简述

搜索当前已注册工具目录，返回最适合任务的工具名、用途和权限摘要。动态工具模式下，返回的工具会在下一轮提示词中展开完整能力卡片。

## 何时使用

不确定该调用哪个工具、当前提示词里没有合适工具、需要查找搜索/读取/编辑/运行/外部 API 等能力时使用。用用户原始意图、错误文本、路径、技术名或目标动作作为 query。

## 不要使用的情况

已经明确知道要调用的工具且该工具已经出现在工具目录中时，不要重复搜索。普通最终回复不需要调用本工具。

## 输入

- `query`：任务或能力查询。
- `includeLoaded`：是否包含已加载工具。

## 输出

返回匹配工具列表：工具名、标题、摘要、适用场景、分数、命中词、匹配能力、推荐原因和权限。下一轮只调用最匹配的具体工具。

## 执行约束

本工具只检索工具目录，不读取项目源码、不访问网络、不执行命令。参数以 JSON 对象表达，由 Senera 运行时、BAML 工具编译器或 Pi tool call 承载。
