# Execution Resource Write

## 简述

向当前会话拥有的后台进程 stdin 写入 UTF-8 文本，并正确处理流背压。

## 何时使用

交互式进程等待回答、确认、命令或标准输入数据时使用。终端式输入通常设置 `appendNewline=true`。

## 不要使用的情况

不要向未知协议的进程盲目写入，不要用它传递超过系统输入预算的大块数据，不要向已结束资源写入。

## 输入

`resourceId` 指定资源，`input` 是文本，`appendNewline` 按当前平台追加换行。

## 输出

写入完成后返回资源快照；返回表示数据已被进程输入流接受，而不是进程已经处理完成。

## 执行约束

输入大小受集中配置限制；写入只允许资源所有者执行，流背压解除前工具不会报告成功。
